CLOUDY

A community networking cloud in a box.

Clommunity project
Què és Cloudy ?

Cloudy és una distribució destinada als usuaris de Xarxes Comunitàries per fomentar la transició i l'adopció de l'ecosistema del cloud
  • Basada en Debian GNU/Linux 8 Jessie
    •
  • Per a diferents arquitectures (x86, amd64, arm...)
    •
  • Per a dispositius GRANS i petits
    •

Què aporta de nou Cloudy ?

  • Entorn web "amigable" per instal·lar i configurar serveis
    •
  • Sistema de descoberta i publicació de serveis dinàmic distribuït i descentralitzat (DADS)
    •
  • Integració amb la web de Guifi
    •

Com instal·lar Cloudy ?

  • Utilitzant una ISO del projecte.
    •
  • Cloudinitzant una debian.
    •
